Process Interaction and CPU Simulation

This project aims to enhance my understanding of key operating system concepts such as cycle instruction, memory, CPU operations, process forking, and inter-process communication via pipes. By simulating separate processes for the CPU and memory, the project provides a hands-on approach to observing how processes interact and coordinate through the CPU and memory. Despite its simplicity, the project effectively covers the fundamental systems within a processor, offering practical insights into how an operating system functions and manages multiple processes.

Networks- Hangman-Multiplayer

This program is a server-client application that simulates a hangman game using FIFOs (named pipes) for secure and managed communication. The server reads motivational quotes from a file and sends a random one to the client upon request. The client, through its own FIFO, communicates with the server to receive quotes. This pipeline-based approach ensures a secure exchange of data between the user and the client.
